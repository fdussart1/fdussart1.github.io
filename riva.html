Riva Iseo

Luxury Yacht - A Class Modeling

Project Presentation

The project was realized during my first year master's degree at the UTBM in a course specialised in 3D and A class modeling. We achieve with sucess (A grade) this project in 4 months in a team of 2. My role was to design the exterior hull and the board of the boat. We used Catia V5 to design, Keyshot and 3DS max to render.

Open Yacht of 8,24m (27 feet), Riva Iseo's design is totaly inspired from the legendary Aquarama. Its technical caracteristics are the following:

  • Length HT: 8,24 m
  • Length: 8,21 m
  • Width: 2,50 m
  • Weight: 2 560 kg (hull only)
  • Power: 320 ch
  • Max Power: 380 ch
  • Transmission: Inboard/Z drive
  • Approval: B/6 pers.
  • Start price: 336 000 €

3D Modeling

The first phase was to create main surfaces from blueprints. For that, we used generative shape design Catia V5's workshop at 95% and ICEM Shape design at 5%.

Surface Quality Checking

We used some tools to check if the surfaces we made was good, well connected or well curved.


  • First tool: Surface Curvature Analysis
    • Each color change means an inflection change.


  • Second tool: isophotes analysis
    • Isophotes are variable black stripes applied to the selected surface which is considered as reflective, and used for analysis purposes. It is a tool for the analysis of the element state.


  • Third tool: Curvature Analysis - Bending comb
    • Each color change means an inflection change.
